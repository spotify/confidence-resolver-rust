# Changelog

## [Unreleased]

### Added

- Initial release of Confidence OpenFeature Provider for Python
- WASM-based local flag resolution
- Async/await support with asyncio
- Automatic state fetching from CDN with ETag caching
- Periodic log flushing to Confidence backend
- Support for boolean, string, integer, float, and object flag types
- Nested flag value access with dot notation
- Configurable state fetch and log flush intervals
- Integration with OpenFeature Python SDK
- Comprehensive test suite (unit, integration, and e2e tests)
